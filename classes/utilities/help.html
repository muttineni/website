Cycle Preparation Help

Student Registration Information

  • Purpose: Generates a list of all students registered for a course in the selected cycle.
  • Instructions for Use: Select a cycle and click submit; click print to print the results of the query.

Course Mailing List E-Mail Addresses

  • Purpose: Generates a list of all mailing lists (listserv) for all courses in the selected cycle.
  • Instructions for Use: Select a cycle and click submit; copy the results into your e-mail program and click send; your message will be sent to all students taking a course during the selected cycle.

Facilitator E-Mail Addresses

  • Purpose: Generates a list of facilitator e-mail addresses for all courses in the selected cycle.
  • Instructions for Use: Select a cycle and click submit; copy the results into your e-mail program and click send; your message will be sent to all facilitators who are teaching a course during the selected cycle.

Facilitator Information

  • Purpose: Generates a list of all facilitators (along with their e-mail address, username, and password) for all courses in the selected cycle.
  • Instructions for Use: Select a cycle and click submit; copy the results into a text file; print or save as needed.

Cycle Preparation Checklist

  • Purpose: Generates the cycle preparation checklist which lists all tasks to be done when preparing for a new cycle.
  • Instructions for Use: Select a cycle and click submit; copy the results into a text file; edit course and cycle codes as indicated.

List Serv Add Commands

  • Purpose: Generates the commands that will add users to the sympa lists.
  • Instructions for Use: Select a cycle and click submit; e-mail the results to sympa@lists.udayton.edu to add all new users to the lists.

.htaccess Usernames and Passwords

  • Purpose: Generates the usernames and encrypted passwords for all users in a given (including the facilitator).
  • Instructions for Use: Select a course and click submit; copy the results and paste them into the .htpasswd file for the course selected.

List Serv Delete Commands

  • Purpose: Generates sympa list serv delete commands based on the response from a review command.
  • Instructions for Use: Copy the review commands from the text box and send them to sympa@lists.udayton.edu; paste the entire response message into the text box and click submit; e-mail the results to sympa@lists.udayton.edu to delete all users (except Sr. Angela Ann) from all lists.

Format Articles for New Course Template

  • Purpose: Formats article text for the new course template.
  • Instructions for Use: Paste the entire text of the article (including HTML) into the text box; click submit; copy the reformatted text and insert it into the database using the admin interface.
